## REXX code to invoke skeleton

```rexx
/* REXX */
/* REXX TO PERFORM FILE TAILORING USING SKELETONS */
USR = 'IBMMF'
ADDRESS ISPEXEC
"LIBDEF ISPSLIB DATASET ID('SKEL.DATASET')"
"FTOPEN TEMP"
"FTINCL SKELMEM"
"FTCLOSE"
"VGET (ZTEMPF) SHARED"
"VIEW DATASET('"ZTEMPF"')"

EXIT 0
```
