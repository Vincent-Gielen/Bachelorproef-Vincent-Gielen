Het idee voor deze bachelorproef is er gekomen door een combinatie van verschillende zaken. Ten eerste omdat ik de scripting taal Rexx aangenaam vind om mee te werken, maar ook door mijn interesse in automatisatie, en de grote focus in de mainframe industrie op modernisatie en het tekort aan werkkrachten.

Ik wil graag Arthur Coucke bedanken voor het op gang helpen van mijn bachelorproef. Het idee om het genereren van JCL te gebruiken als vergelijking was oorspronkelijk van Arthur. Uit dit initiële idee vloeiden enkele interessante voorstellen om het hele automatisatie landschap te analyseren. Uiteindelijk heb ik besloten om de vergelijking enkel tussen Rexx en Python te houden. Daarnaast heb ik kunnen rekenen op de feedback van mijn promotor, Leendert Blondeel, om steeds kritisch te blijven kijken naar mijn idee en werk, zodat ik zeker op het juiste spoor bleef tijdens de uitwerking van de bachelorproef.

Om de ideeën voor de PoC vorming te geven heb ik heel veel steun gekregen van mijn co-promotor, Bobby Tjassens. Niet alleen inhoudelijk, maar ook voor de code kon ik doorheen heel het proces op de ervaring van Bobby rekenen. Wekelijkse meetings met toegang tot een systeem om op te werken, en ontelbaar veel code voorbeelden om mij op gang te helpen. Deze bachelorproef was niet mogelijk geweest zonder de hulp van Bobby.

Een groep die ik zeker niet mag vergeten bedanken, is de mainframe community, zowel op Discord als op Linkedin. Ik heb enorm veel input en feedback gekregen op elke vraag die ik heb gesteld. Ook het aantal antwoorden op mijn enquête heeft me enorm vooruit geholpen om de beste requirements en use cases te bepalen. Ook wil ik graag Emma Skovgård bedanken voor haar tijd en input voor het interview, en daarnaast ook voor altijd input te geven over manieren waarop Python kan worden gebruikt op het mainframe.

Als laatste wil ik ook mijn zus bedanken, die de structuur en taal van mijn bachelorproef enorm veel heeft vooruitgeholpen en verbeterd.
